== Description ==
Безкоштовний плагін XPAY Payment GateWay дозволяє інтегрувати прийом платежів через систему XPAY в магазинах, що розроблені на платформі WooCommerce. 

= Функціональні можливості плагіну: =
*	Проведення налаштування на тестовому середовищі XPAY з можливістю переключення на продуктове середовище;
*	Управління параметрами оплати без необхідності програмування; 
*	Вибір ідентифікатора клієнта при проведенні оплати (номер мобільного телефону чи пошта);
*	Налаштування переадресації браузера клієнта після виконання оплати на визначену сторінку; 
*	Прийом postback із успішною оплатою;
*	Відображення актуального статусу оплати.

==Installation==
= Крок 1 =
Для інсталяції плагіну необхідно перейти у розділ [“Плагіни”](https://wordpress.org/plugins/), виконати пошук за назвою “XPAY Payment GateWay” та натиснути кнопку “Download” для завантаження ZIP файлу з відповідним ім'ям (XPAY Payment GateWay).

= Крок 2 =
Після отримання файлу необхідно в панелі адміністратора вашого сайту перейти у розділ “Плагіни” обравши підрозділ “Додати новий” та натиснувши кнопку “Завантажити плагін”.


Слідуючим кроком є активація плагіну. Для цього необхідно перейти у розділі “Плагіни” до підрозділу “Встановлені” та знайти серед переліку XPAY Payment GateWay.

= Крок 3 =
Активація плагіну може бути також проведена безпосередньо під час завантаження файлу, через кнопку “Активувати плагін”, що знаходится в наступному повідомленні після успішного завантаження. 

= Крок 4 =
Після активації плагіну переходимо у розділ “WooCommerce” та обираємо підрозділ “Налаштування”. Серед варіантів налаштування обираємо “Платежі”, та знаходимо у переліку метод “XPAY” після чого активуємо чек-бокс.


Для використання плагіну та можливості прийому платежів необхідно завершити конфігурацію, натиснувши кнопку “Управління” та перейшовши до параметрів.

= Крок 5 =
Параметри:
*	Title:  назва методу оплати на сайті; 
*	Description: коротка назва методу оплати на сайті; 
*	XPAY Service URL:  
Тестове середовище: https://stage-mapi.xpay.com.ua/uk/frame/widget/banner-payment 
Продуктове середовище: https://mapi.xpay.com.ua/uk/frame/widget/banner-payment 
*	XPAY Public Key: публічний ключ XPAY продуктового чи тестового середовища для перевірки підпису; 
*	Open in new windows: дозволяє відкривати сторінку оплати у новому вікні
*	Partner ID: ідентифікатор партнера у системі XPAY;
*	User identified by: вибір ідентифікатору користувача при передачі платежу, один із варіантів - електронна пошта або номер телефону;
*	Return URL: вибір із переліку сторінок магазину на яку буде переадресовано браузер клієнта після здійснення оплати; 
*	or manual link: дозволяє вказати URL будь-якого ресурсу для переадресації браузер клієнта після здійснення оплати; 
*	Show Payment info:  вказує на необхідність відображення деталізованої інформації про платіж у віджеті;
*	Payment item caption: вибір поля із переліку варіантів щодо інформації яка передається ро товар чи послугу;
*	Service URL for Pay request: URL обробника postback запиту pay на стороні магазину. Запит pay надсилається системою XPAY після успішного проведення платежу. Перед початком роботи необхідно надати в XPAY URL тестового та продуктивного середовища магазину.

Після виконання всіх налаштувань для клієнта при оформленні замовлення на сайті буде відображатись варіант оплати через XPAY. 


>**Натискаючи кнопку “Оформити” клієнт буде переходити до платіжної сторінки з деталями замовлення та реквізитами оплати.**


>**Інформація про актуальний статус оплати відображається у розділі “Замовлення”.**

==About XPAY==
XPAY (ТОВ “ІКС ПЕЙ ГРУП”) - PSP (Payment Service Provider) провайдер, що здійснює процесинг платежів і надає послуги широкому спектру українських учасників ринку електронної комерції.

XPAY відповідає усім запитам індустрії та стандарту PCI DSS level 1 v3.2., який щорічно підтверджується міжнародним аудитом.
